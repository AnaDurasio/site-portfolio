Como Rodar o Projeto

Clone este repositório:

git clone https://github.com/AnaDurasio/site-portfolio.git

Acesse a pasta do projeto:

cd site-portfolio

Abra o arquivo index.html no navegador ou utilize um servidor local.

Deploy
